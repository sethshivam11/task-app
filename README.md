# Todo App

This is a simple project built in MERN stack and typescript with tailwindcss, shadcn/ui.

## Clone this project

`git clone https://github.com/sethshivam11/todo.git`

## Run this project

`npm run build`

`npm start`

## Environment Variables

`PORT`

`NODE_ENV` -- expected `production`

`MONGODB_URI`

`ACCESS_TOKEN_SECRET`

`ACCESS_TOKEN_EXPIRY`
